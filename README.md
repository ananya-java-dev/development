# To-Do List App

A simple To-Do List application built using HTML, CSS and JavaScript.

## Features
- Add tasks
- Mark tasks as completed
- Delete tasks
- Filter tasks (All, Active, Done)
- Clear completed tasks
- Local storage support

## Technologies Used
- HTML
- CSS
- JavaScript
